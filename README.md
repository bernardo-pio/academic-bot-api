   # Academic Research Bot API

   API gratuita para busca de artigos científicos em múltiplas fontes.

   ## Fontes disponíveis:
   - Google Scholar
   - PubMed
   - arXiv
   - Semantic Scholar
   - OpenAlex

   ## Como testar localmente:
```bash
   pip install -r requirements.txt
   uvicorn main:app --reload
```

   Acesse: http://localhost:8000/docs
```
